# Compilador-da-Linguagem-Jack
Este repositório contém o desenvolvimento de um compilador para a linguagem Jack, como parte das atividades práticas do curso. O objetivo é traduzir o código-fonte Jack para bytecode da Máquina Virtual (VM).
